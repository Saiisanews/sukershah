About

Nama saya Sukersah, saya sering menuliskan nama saya dengan sebutan Sukershah hingga saat ini. Saya lahir di Lombok Utara dan dibesarkan di keluarga sederhana. Awal saya merantau ke Jakarta pada tahun 2007 untuk menuntut ilmu dibidang Teknologi Informasi. Sebenarnya ketertarikan saya awalnya pada ilmu sains dan sempat mendaftar SPMB di Universitas Mataram dan lulus namun tidak saya ambil, karena pertimbangan pertimbangan pekerjaan ke depannya, jadi saya memilih untuk mengambil jurusan Komputer di salah satu Sekolah Tinggi Ilmu Komputer di Jakarta. Sekian perkenalan singkat dari saya.

Website Portofilo ini adalah tempat untuk menampilkan hasil karya saya yang saaat ini dalam proses pengembangan. Saya berharap daftar portofolio yang saya tampilkan tampilkan ini dapat menjadi bahan pertimbangan dalam proses rekruitmen untuk posisi sebagai Software Engineer kedepannya.
